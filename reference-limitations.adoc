= Limitações no NetApp Copy and Sync
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Limitações conhecidas identificam plataformas, dispositivos ou funções que não são suportados por esta versão do produto ou que não interoperam corretamente com ele. Revise essas limitações cuidadosamente.

O NetApp Copy and Sync não é compatível nas seguintes regiões:

* Regiões governamentais da AWS
* Regiões do Governo do Azure
* China
